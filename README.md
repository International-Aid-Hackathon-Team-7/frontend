# Bridge App

A full-stack single-page MERN web application with full CRUD capabilities built for Canadian users to learn and share information about climate change.

## Technologies Used
- JavaScript
- NodeJS
- Express
- React
- React-Bootstrap
- React-router-dom
- CSS3
- MongoDB
- Mongoose
- Google Firebase
- Axios
- AWS SDK
